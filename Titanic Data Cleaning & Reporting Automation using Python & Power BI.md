# Data-Cleaning-Reporting-Automation

📊 Titanic Data Cleaning & Reporting Automation using Python & Power BI

This project focuses on cleaning, preprocessing, and reporting the Titanic dataset using Python and Power BI. The dataset was cleaned by handling missing values and preparing it for analysis. An interactive Power BI dashboard was then built to visualize passenger demographics, survival patterns, fare distribution, and embarkation details.

🔹 What I Implemented
Imported and explored the Titanic dataset using Pandas
Performed data cleaning by handling missing values
Checked for duplicate records
Prepared the cleaned dataset for reporting
Built an interactive Power BI dashboard
Created Card visuals for Total Passengers, Total Survivors, Survival Rate, and Average Age
Visualized Survival by Gender and Passenger Class
Analyzed Passenger Distribution by Age Group
Compared Average Fare across Passenger Classes
Visualized Passenger Distribution by Embarkation Port
Added interactive slicers for Gender, Passenger Class, Embarkation, and Survival Status
Added a Key Insights section to summarize important findings
📚 What I Learned
Data cleaning and preprocessing using Pandas
Handling missing values and preparing data for reporting
Building interactive dashboards in Power BI
Creating charts, cards, slicers, and business insights
Using interactive reports to support data-driven analysis
🛠 Tools & Technologies

Python | Pandas | Power BI | DAX

📈 Key Outcome

Successfully cleaned and prepared the Titanic dataset for analysis and built an interactive Power BI dashboard that enables users to explore passenger demographics, survival trends, fare analysis, and embarkation details through dynamic visualizations.
